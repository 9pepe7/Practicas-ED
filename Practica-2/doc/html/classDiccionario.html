Descripción detallada

T.D.A. Diccionario.

Una instancia del tipo de datos abstracto Diccionario es un objeto que contiene un array variable de Terminos del T.D.A. Termino, incluido en este archivo mediante el include "Termino.h", y un valor entero llamado tamanio, que indica el número de elementos del vector de Terminos. Lo representamos como una lista con todos los Terminos.
