T.D.A. Cola_max.

Una instancia del TDA Cola_max es, a efectos prácticos, una cola (FIFO) de la que además se puede saber el máximo. Esta implementación contiene dos vectores, uno con sus miembros y el otro para el máximo. El primero actuará como un queue simple con sus miembros en orden. El segundo irá añadiendo los nuevos elementos también, pero además, será modificado si se añade un nuevo máximo.

Una instancia del TDA Cola_max es, a efectos practicos, una cola (FIFO) de la que además se puede saber el máximo. Esta implementación contiene dos pilas con sus miembros repartidos entre ambas, y una variable que contiene dicho máximo, que se modificará cuando se añada un maximo nuevo, o cuando se elimine dicho maximo.
